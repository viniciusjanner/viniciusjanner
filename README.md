# Sobre mim

Desde 2014, desenvolvendo aplicativos Android com Java e Kotlin.

Aplicando boas práticas em projetos modularizados, robustos e internacionais.

Utilizando clean architecture, padrão MVVM e injeção de dependência (Hilt e Koin).

Consumindo serviços de API com Retrofit e Coroutines.

Realizando integrações contínuas com Azure, GitHub Actions e Sonar Cloud.

Sempre atuando com times multidisciplinares em ambiente ágil com Kanban e Scrum.
